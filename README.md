🎓 AI Academic Intelligence System

An end-to-end Machine Learning project that predicts student academic performance using Linear Regression and deploys the model through an interactive Streamlit dashboard.
Developed as part of my Meachine learning Internship at Elevvo.

📌 Overview

This application predicts a student's exam performance based on:

- Study Hours  
- Attendance Percentage  
- Mental Health Rating  
- Sleep Hours  
- Part-Time Job Status  

The model outputs a predicted score between 0% and 100%.

🚀 Features

- Real-time performance prediction  
- Interactive gauge visualization  
- Feature importance analytics  
- Clean enterprise-style dashboard  

 🛠 Technologies Used
Python • Streamlit • Scikit-Learn • NumPy • Plotly • Joblib  

⚙️ Run the Project

1. Install dependencies  
   pip install -r requirements.txt  

2. Run the app  
   python -m streamlit run app.py
